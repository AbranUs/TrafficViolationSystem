# TrafficViolationSystem

Sistema de gestión y control de infracciones de tránsito (Traffic Violation System).

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

```text
TrafficViolationSystem/
 ├── backend/             # Código fuente del servidor, APIs y lógica de negocio.
 ├── frontend/            # Código fuente de la interfaz de usuario (Web/Móvil).
 ├── docs/                # Documentación del proyecto dividida por fases/áreas:
 │    ├── 01_Gestion_Proyecto
 │    ├── 02_Negocio
 │    ├── 03_Analisis
 │    ├── 04_Arquitectura
 │    ├── 05_Desarrollo
 │    ├── 06_Operacion
 │    └── 07_Gestion_Proyecto
 └── README.md            # Este archivo con la descripción general del sistema.
```
